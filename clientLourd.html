Introduction

L'application TaskMastery est un projet qui permet à l'utilisateur de voir les tâches et ou est ce que chaque tâche en est dans son avancement. L'application permet de visualiser chaque projet avec tableau de chaque tâche et de leurs avancement. Il est possible de créer des étiquettes de les modifier ou les supprimer, il est possible de voir les participants via un chat avec les participants. Les tâches sont stockées dans une base de données MySQL en ligne sur le server et le code est en WPF (dotnet).

Prérequis

  1. Conception et Archicture :
    • J'ai maîtrisé la méthode MVVM (Modèle Vue VueModèle) pour organiser efficacement le code.
    • J'ai appris à utiliser le framework WPF, qui facilite le développement d'application de bureau avec dotnet.
  2. Gestion de Base de Données :
    • J'ai acquis des connaissances en SQL pour concevoir et interroger la base de données.
    • J'ai utilisé MySQL Workbench pour la modélisation et la gestion de la base de données.
  3. Contrôle de Version :
    • J'ai utilisé Git pour le versionnement du code, permettant un suivi efficace des modifications et une collaboration aisée.
    • J'ai utilisé GitHub pour stocker le code et faciliter le partage. Pour pouvoir travailler sur plusieurs machines.
  4. Environnement de Développement :
    • Pour l'écriture du code, j'ai utilisé Visual Studio, un éditeur de texte puissant et polyvalent.
    • J'ai également exploité GitHub pour le stockage et la collaboration sur le code source.
    • J'ai utilisé XAMPP pour pouvoir tester la base de données.
    • J'ai utilisé MySQL Workbench pour la modélisation et la gestion de la base de données.
    • J'ai utilisé Looping pour le MLD et le MCD de la base de données.
  5. Front-End :
    • Avant de commencer j'ai du faire le template sur figma pour avoir une idée de ce que je voulais.
    • Pour le design et l'interactivité, j'ai utilisé WPF pour créer une interface utilisateur moderne et ergonomique en XAML.

Installation

Pour installer TaskMastery, il suffit de télécharger le fichier d'installation et de suivre les instructions à l'écran. Il est possible de personnaliser l'installation en choisissant les composants à installer. Vous pouvez télécharger le projet tout en bas de la page.

Déploiement

Pour déployer TaskMastery, il suffit de copier les fichiers sur le serveur et de configurer les paramètres de connexion à la base de données. Il est possible de personnaliser l'interface en modifiant les fichiers de configuration.

Utilisation

Pour utiliser TaskMastery, il suffit de se connecter avec un compte utilisateur ou de s'inscrire. Ensuite vous pouvez avoir tous les projets et lorsque vous cliquez sur un projet vous avez le tableau des tâches et vous pouvez voir les participants dans un second tableau lorsque vous cliquez sur la tâche. Il y a également la liste des étiquettes que vous pouvez ajouter une étiquette ou modifier ou supprimer. L'application permet également la modification ou la suppression du compte.
