Introduction

TaskMastery est un projet de gestion de tâches. Il permet de créer, modifier, supprimer et visualiser des tâches. Il est possible de participer à des tâches, il est possible de parler via un chat avec les participants. Les tâches sont stockées dans une base de données MySQL et le code est en Laravel (PHP)

Prérequis

  1. Conception et Archicture :
    • J'ai maîtrisé la méthode MVC (Modèle Vue Contrôleur) pour organiser efficacement le code.
    • J'ai appris à utiliser le framework Laravel, qui facilite le développement web avec PHP.
  2. Gestion de Base de Données :
    • J'ai acquis des connaissances en SQL pour concevoir et interroger la base de données.
    • J'ai utilisé MySQL Workbench pour la modélisation et la gestion de la base de données.
  3. Contrôle de Version :
    • J'ai utilisé Git pour le versionnement du code, permettant un suivi efficace des modifications et une collaboration aisée.
    • J'ai utilisé GitHub pour stocker le code et faciliter le partage. Pour pouvoir travailler sur plusieurs machines.
  4. Environnement de Développement :
    • Pour l'écriture du code, j'ai utilisé Visual Studio Code, un éditeur de texte puissant et polyvalent.
    • J'ai également exploité GitHub pour le stockage et la collaboration sur le code source.
    • J'ai utilisé XAMPP pour lancer un serveur web local et tester le code PHP.
    • J'ai utilisé MySQL Workbench pour la modélisation et la gestion de la base de données.
    • J'ai utilisé Looping pour le MLD et le MCD de la base de données.
  5. Front-End :
    • Avant de commencer j'ai du faire le template sur figma pour avoir une idée de ce que je voulais.
    • Pour le design et l'interactivité, j'ai employé Bootstrap pour le CSS, offrant une base solide pour un design réactif et attrayant.
    • J'ai intégré jQuery pour simplifier le JavaScript et manipuler le DOM de manière efficace.
  6. Requêtes Asynchrones :
    • J'ai appris à utiliser AJAX pour effectuer des requêtes asynchrones, améliorant ainsi la réactivité et l'expérience utilisateur.
  7. Gestion des Dépendances :
    • J'ai utilisé npm pour gérer les dépendances en CSS et JavaScript, facilitant ainsi l'intégration et la mise à jour des bibliothèques externes.
  8. Migration et Seeder :
    • J'ai maîtrisé l'utilisation des migrations et des seeders dans Laravel pour gérer facilement l'évolution de la structure de la base de données et pour générer des données de test.
  9. Gestion de Sessions et Cookies :
    • J'ai acquis des compétences dans l'utilisation des sessions et des cookies pour la gestion des projets, assurant ainsi une expérience utilisateur personnalisée et sécurisée.

Installation

Pour installer TaskMastery, il suffit de cloner le dépôt GitHub sur votre machine locale. Ensuite, il faut installer les dépendances avec npm et composer. Enfin, il faut configurer le fichier .env avec les informations de connexion à la base de données. Pour finir, il faut exécuter les migrations et les seeders pour initialiser la base de données avec des données de test.

Déploiement

Pour déployer TaskMastery, il suffit de transférer les fichiers sur un serveur web. Il faut également importer la base de données MySQL sur le serveur. Enfin, il faut configurer le fichier .env avec les informations de connexion à la base de données sur le serveur.

Utilisation

Pour utiliser TaskMastery, il suffit de se connecter avec un compte utilisateur ou de s'inscrire. Tous d'abord il faut créé un projet pour pouvoir créer les tâches à l'intérieur, Il est possible de créer des tâches, de les modifier, de les supprimer, de les visualiser. Il est possible de participer à des tâches et de discuter avec les participants via un chat. Mais aussi d'ajouter des étiquettes pour pouvoir ce retrouver plus facilement.
